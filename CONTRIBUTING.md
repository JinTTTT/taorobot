# Contributing to taorobot

Thanks for being here. 🤖

`taorobot` is a **learn-in-public** project. You don't need to be an expert to
take part — if you're learning robotics too, you're exactly who this is for.
No contribution is too small.

## Ways to contribute

- 💬 **Ask a question** — open an issue. If it confused you, it confused others.
- 🐛 **Report a bug** — what you did, what you expected, what happened instead.
- ✍️ **Improve the docs** — clearer explanations are hugely valuable here.
- 🧑‍💻 **Write code** — fix a bug, add a test, or tackle a limitation listed in a package README.

## Making a change

1. **Fork** the repo and clone your fork.
2. **Create a branch:** `git checkout -b my-change`.
3. **Make your change**, then build and test:
   ```bash
   colcon build --symlink-install && source install/setup.bash && colcon test
   ```
4. **Commit** with a clear message and **open a pull request** against `master`.
5. In the PR, say what you changed and how you tested it.

Don't worry about getting it perfect — we'll work through review together.

## Reporting bugs

Open an [issue](https://github.com/JinTTTT/taorobot/issues) with the command you
ran, what you expected, what happened, any error text, and your ROS 2 / OS version.

Be kind — everyone here is learning, including the maintainer. 🚀
